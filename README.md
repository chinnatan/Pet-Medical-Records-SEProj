# Pet-Medical-Records-2018
### เวชระเบียนสัตว์เลี้ยง
```
  ปัจจุบันความนิยมในการเลี้ยงสัตว์ มีความนิยมที่แพร่หลาย และกำลังเพิ่มมากขึ้นเรื่อยๆ ทำให้มีคลินิก หรือโรงพยาบาลสัตว์เลี้ยงมากขึ้น 
แต่เนื่องด้วยแต่ละโรงพยาบาลไม่มีการทำงานร่วมกัน ทำให้เกิดปัญหาบ่อยครั้งในการให้บริการสัตว์เลี้ยงโดยเมื่อผู้ดูแลสัตว์เลี้ยงต้องการที่
จะย้ายสัตว์เลี้ยงไปโรงพยาบาลใหม่ หรือไม่สะดวกในโรงพยาบาลเดิม โดยโรงพยาบาลได้ทำการแก้ไขโดยการจดบันทึกรายละเอียดคร่าวๆ
ของการรักษาให้กับผู้ดูแลสัตว์เลี้ยงไว้ในเอกสารข้อมูลสัตว์เลี้ยง ประกอบด้วย รายการวัคซีนที่ฉีดเรียบร้อยแล้ว และยังไม่ได้ฉีด วันเวลานัด 
และข้อมูลสัตว์เลี้ยงคร่าวๆ ซึ่งไม่ได้เป็นข้อมูลการรักษาแบบละเอียดอีกทั้งง่ายต่อการสูญหาย
  ทางเราจึงอยากรวมข้อมูลต่างๆ ไว้เป็นที่เดียวกัน เพื่อให้ง่ายต่อการใช้งาน ในการเก็บข้อมูลสัตว์เลี้ยง และไม่สูญหาย 
โดยเป็นลักษณะ android application ที่มีชื่อว่า "Pet Medical Records"
```
